# Changelog

## v1.0.0 - 2026-05-19

首个正式发布版本。

- 发布 `cgx-baseos` 主入口技能，用于判断用户应进入诊断、构建还是维修路径。
- 发布 `cgx-baseos-diagnose`，用于业务数据诊断、AI 数仓判断和一页业务数据地图输出。
- 发布 `cgx-baseos-build`，用于按基础数据层、录入与联动层、展示看板层、验收发布层构建或重构经营数仓。
- 发布 `cgx-baseos-repair`，用于维修已有飞书 Base、多维表格、字段、公式、lookup、视图、看板、筛选器和工作流问题。
- 内置销售系统、一人公司、企业服务、经营财务四类阶段化模板骨架。
- 内置构建脚本，可生成四个独立 skill zip 和统一整包。
