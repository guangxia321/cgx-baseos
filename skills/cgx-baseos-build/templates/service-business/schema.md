# 企业服务经营系统模板

## 适合谁

适合咨询、代运营、软件服务、培训、设计、营销、定制开发等企业服务团队。第一版优先跑通：

线索 -> 商机 -> 方案/报价 -> 合同/订单 -> 项目交付 -> 收款 -> 续费 / 客户成功

## 阶段 1：基础数据层

| 层级 | 表 | 作用 |
| --- | --- | --- |
| 主数据 | 客户管理 | 公司、联系人、行业、来源、客户等级 |
| 主数据 | 服务产品 / 套餐 | 服务类型、标准价格、交付周期、成本口径 |
| 事实 | 线索 | 来源、需求、负责人、跟进状态 |
| 事实 | 商机 | 预计金额、阶段、预计成交时间、关联客户 |
| 事实 | 方案 / 报价 | 报价版本、服务范围、报价金额、状态 |
| 事实 | 合同 / 订单 | 合同金额、签约日期、服务周期、付款节点 |
| 事实 | 项目交付 | 项目、里程碑、任务、交付物、状态 |
| 事实 | 收款记录 | 合同/项目回款、凭证、渠道 |
| 事实 | 成本 / 财务流水 | 外包、人工、广告、工具、其他成本 |

## 阶段 2：录入与联动层

日常录入页：

- 销售录入线索、商机、报价、合同。
- 交付录入项目、里程碑、任务、交付物。
- 财务录入收款、成本、凭证。
- 客户成功录入续费状态和客户反馈。

自动字段：

- 自动月份
- 合同累计回款
- 项目累计成本
- 项目利润
- 回款进度
- 任务完成率
- 续费到期时间标签

显式状态字段：

- 线索状态：新线索 / 跟进中 / 已转商机 / 无效
- 商机阶段：初沟通 / 需求确认 / 报价中 / 谈判中 / 已成交 / 已丢单
- 合同状态：待签 / 已签 / 执行中 / 已完成 / 终止
- 项目状态：未开始 / 进行中 / 风险中 / 已完成 / 暂停
- 结算状态：未结清 / 部分结清 / 已结清 / 退款中
- 续费状态：未到期 / 待续费 / 已续费 / 流失

工作流：

- 商机成交后创建合同/项目草稿。
- 合同创建后生成付款节点。
- 项目里程碑到期提醒。
- 收款后更新结算状态。
- 续费到期提醒。

## 阶段 3：展示看板层

| 组件 | 数据源 | 指标字段 | 筛选字段 |
| --- | --- | --- | --- |
| 销售漏斗金额 | 商机 | 预计金额 | 商机阶段、负责人 |
| 本月签约额 | 合同 / 订单 | 合同金额 | 月份筛选 |
| 本月回款 | 收款记录 | 收款金额 | 月份筛选 |
| 未收款 / 逾期应收 | 合同 / 订单 | 未收款金额 | 结算状态、是否逾期 |
| 项目状态分布 | 项目交付 | 项目数 | 项目状态 |
| 风险项目列表 | 项目交付 | 风险等级/逾期任务 | 项目状态、负责人 |
| 续费提醒 | 合同 / 客户成功 | 到期时间 | 续费状态 |
| 项目利润 | 项目交付 / 成本流水 | 项目利润 | 月份筛选、项目状态 |

## 阶段 4：验收发布层

- 能从客户看到线索、商机、合同、项目和收款。
- 能从项目看到交付状态、回款状态和成本。
- 销售、交付、财务读取同一套客户、合同和项目数据。
- 用户不用维护重复客户名、项目名或合同快照。
- 看板完整重载后仍渲染，并用两个不同月份或项目状态验证指标变化。
