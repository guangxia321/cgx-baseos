# 一人公司经营系统模板

## 适合谁

适合创始人自己负责获客、销售、交付、收款和复盘的一人公司或小团队。第一版先跑通：

产品/服务 -> 线索 -> 客户 -> 订单/合同 -> 交付 -> 收款 -> 复购

内容资产和渠道账号可作为扩展模块，不要影响第一版上线。

## 阶段 1：基础数据层

| 层级 | 表 | 作用 |
| --- | --- | --- |
| 主数据 | 产品 / 服务 | 服务包、价格、成本、交付周期、状态 |
| 主数据 | 渠道账号 | 小红书、抖音、视频号、公众号、社群等 |
| 主数据 | 客户管理 | 客户来源、标签、状态、联系人、需求 |
| 事实 | 线索 / 咨询 | 咨询来源、需求、跟进状态、转化结果 |
| 事实 | 订单 / 合同 | 成交服务、金额、交付周期、关联客户 |
| 事实 | 交付记录 | 交付任务、里程碑、交付物、客户反馈 |
| 事实 | 收款记录 | 收款金额、日期、渠道、凭证 |
| 事实 | 财务流水 | 支出、成本、工具订阅、外包费用 |

可选扩展：

- 内容资产：选题、发布时间、渠道、转化产品。
- 复购 / 续费：到期时间、续费状态、复购记录。

## 阶段 2：录入与联动层

日常录入页：

- 新线索：来源、客户、需求、跟进状态。
- 新订单：客户、产品/服务、成交金额、日期。
- 新交付：订单/项目、任务、状态、截止日期。
- 新收款：关联订单、金额、日期、凭证。
- 新支出：金额、科目、日期、凭证。

Agent 入口：

- 按客户查询当前订单、交付、收款和待办。
- 代录聊天里的线索、客户反馈、收款凭证和交付状态。

自动字段：

- 自动月份
- 自动产品/服务价格
- 自动交付状态
- 自动收款状态
- 自动未收款
- 自动利润
- 客户累计成交
- 渠道转化统计

显式状态字段：

- 线索状态
- 交付状态
- 结算状态
- 复购/续费状态
- 月份筛选

## 阶段 3：展示看板层

| 组件 | 数据源 | 指标字段 | 筛选字段 |
| --- | --- | --- | --- |
| 本月成交额 | 订单 / 合同 | 成交金额 | 月份筛选 |
| 本月回款 | 收款记录 | 收款金额 | 月份筛选 |
| 未收款 | 订单 / 合同 | 自动未收款 | 结算状态 |
| 待交付 / 逾期交付 | 交付记录 | 任务数或状态 | 交付状态、截止日期 |
| 产品/服务销售排行 | 订单 / 合同 | 成交金额 | 月份筛选、产品/服务 |
| 渠道来源转化 | 线索 / 咨询 | 转化结果 | 渠道、线索状态 |
| 月度利润 | 财务流水 / 订单 | 自动利润 | 月份筛选 |

## 阶段 4：验收发布层

- 用户能从一个订单看到客户、服务、交付、收款、利润。
- Agent 能按客户或订单查询当前交付和收款状态。
- 用户不用为了看月报手工维护汇总表。
- 日常录入页足够短，不展示后台公式和 lookup 字段。
- 看板完整重载后仍渲染，并用两个不同月份验证指标变化。
