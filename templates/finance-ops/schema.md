# 经营财务模板

## 适合谁

适合已经有订单、项目、收款或支出记录，但看不清收入、支出、利润、未收款和凭证的小企业。它解决的是经营财务，不替代专业会计系统。

第一版跑通：

订单 / 项目 -> 收款 -> 支出 / 成本 -> 利润 -> 月度经营看板

## 阶段 1：基础数据层

| 层级 | 表 | 作用 |
| --- | --- | --- |
| 主数据 | 财务科目 | 收入、成本、费用、退款、调整等分类 |
| 主数据 | 账户 / 渠道 | 对公账户、微信、支付宝、银行卡等 |
| 事实 | 收款记录 | 每一笔真实收款，保留凭证 |
| 事实 | 财务流水 | 支出、费用、非订单收入、退款、调整 |
| 事实 | 订单 / 项目 | 财务记录关联的业务来源 |
| 派生 | 月度经营汇总 | 月度收入、支出、利润、未收款 |

## 阶段 2：录入与联动层

日常录入页：

- 收款记录：收款日期、关联订单/项目、客户、收款金额、收款渠道、凭证附件、备注。
- 财务流水：发生日期、类型、财务科目、金额、关联订单/项目/客户、支付渠道、凭证附件、备注。

批量录入页：

- 支持按日期、科目、金额、渠道、凭证批量导入历史流水。
- 导入后校验金额为空、日期异常、科目未匹配、凭证缺失、重复流水。

自动字段：

- 自动月份
- 自动收入金额
- 自动支出金额
- 自动经营利润
- 自动未收款金额
- 项目累计收入
- 项目累计成本
- 项目利润

显式状态字段：

- 结算状态：未结清 / 部分结清 / 已结清 / 退款中
- 是否逾期
- 凭证状态：未上传 / 已上传 / 待补充
- 入账状态：待确认 / 已确认 / 需调整
- 月份筛选

## 阶段 3：展示看板层

| 组件 | 数据源 | 指标字段 | 筛选字段 |
| --- | --- | --- | --- |
| 本月收入 | 收款记录 / 财务流水 | 收入金额 | 月份筛选 |
| 本月支出 | 财务流水 | 支出金额 | 月份筛选、财务科目 |
| 本月经营利润 | 月度经营汇总 | 经营利润 | 月份筛选 |
| 未收款金额 | 订单 / 项目 | 自动未收款金额 | 结算状态 |
| 逾期应收 | 订单 / 项目 | 自动未收款金额 | 是否逾期 |
| 费用分类排行 | 财务流水 | 金额 | 财务科目、月份筛选 |
| 客户回款排行 | 收款记录 | 收款金额 | 客户、月份筛选 |
| 月度收入 / 支出 / 利润趋势 | 月度经营汇总 | 收入、支出、利润 | 月份筛选 |

## 阶段 4：验收发布层

- 新增一笔收款后，对应订单 / 项目的结算状态能更新。
- 新增一笔支出后，本月费用和利润能变化。
- 凭证能从收款或支出记录追溯到客户、订单或项目。
- 月度经营看板不需要用户手工维护。
- 看板完整重载后仍渲染，并用两个不同月份验证收入、支出、利润变化。

## 边界

默认不做复式记账、税务申报、完整总账、发票全生命周期、多公司合并报表。如果用户需要这些能力，先确认是否已有专业财务软件，再判断本模板只做经营看板和数据入口，还是不适合承接。
