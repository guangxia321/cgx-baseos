# Templates

这个目录用于存放可复用的业务模板。

模板可以包括：

- 表结构模板
- 字段清单
- 公式设计
- 看板组件设计
- 工作流设计
- 行业化改造说明

模板应尽量保持工具中立。即使默认落地到飞书多维表格，也要说明背后的业务建模原则，方便迁移到其他低代码表格、SaaS 或轻量 ERP 工具。

## 第一版模板

| 模板 | 用途 |
| --- | --- |
| `sales-system/` | 销售、客户、订单、收款和经营看板 |
| `solo-business/` | 一人公司产品/服务、线索、客户、交付、收款和复盘 |
| `service-business/` | 企业服务线索、商机、合同、项目交付、续费和回款 |
| `finance-ops/` | 经营财务：收款、支出、利润、凭证、月度经营看板 |

每个模板先用 `schema.md` 按四阶段描述：基础数据层、录入与联动层、展示看板层、验收发布层。后续如果沉淀出可直接导入或复制的 Excel / Base 文件，放在对应模板目录内，不单独拆成技能。
